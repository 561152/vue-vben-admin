# Playwright 测试资源

此目录存放 Playwright E2E 测试所需的资源文件。

## 文件说明

- `test-homework.jpg` - 测试用作业图片（需要手动添加）
- `test-homework-large.jpg` - 大文件测试图片 (>3MB)
- `test-homework-invalid.txt` - 无效格式文件测试

## 添加测试图片

请添加一张真实的作业图片（数学题或其他作业）到此目录，并命名为：
- `test-homework.jpg` - 标准测试图片（建议 1-2MB）

## 注意事项

- 不要提交包含个人信息的图片
- 测试图片应该包含清晰的文字内容
- 建议图片大小在 1-3MB 之间
